# 2026-05-05 — R9. Async gateway path

## Roadmap item
**R9. Async gateway path.** Add an async-aware mediation path so callers using
`async def` tools (most production agent stacks, including the real `mcp`
SDK) can route through the gateway without hopping in and out of
`asyncio.run`.

## Acceptance criteria
- Async-aware gateway entry point: `Gateway.aexecute(call, fn, *args, …)` —
  awaits the underlying coroutine and an awaitable audit writer.
- Async tool decorator: `Gateway.wrap_tool_async(fn=None, *, tool_name=…,
  taint_spec=…, resource_arg=…)` mirroring the sync `wrap_tool` API.
- Async MCP adapter: `wrap_mcp_session_async(gateway, session, *, …)` for
  duck-typed async sessions exposing `await session.list_tools()` and
  `await session.call_tool(name, arguments)` (the real `mcp` SDK shape).
- Reserved kwargs (`apg_input_label`, `apg_agent_id`, `apg_call_id`,
  `apg_resource`) behave identically to the sync path and are stripped
  before forwarding.
- DENY / REVIEW raise `PolicyDenied` / `PolicyReview` — the existing
  exception types — so the rest of the codebase keeps a single refusal
  contract.
- Sync path unaffected; existing 253 tests still pass.

## Plan
1. Add `aexecute` and `wrap_tool_async` to `gateway.py`. Reuse `decide()`
   (pure) so async/sync share one decision routine. Audit writer can be
   sync *or* return an awaitable — `aexecute` awaits the result if it is
   awaitable, leaving sync writers unchanged.
2. Add `mcp_async_adapter.py` with `wrap_mcp_session_async`. Mirror the
   sync adapter's resource_arg / prefix / taint_spec semantics.
3. Export the new symbols from the package `__init__`.
4. Tests:
   - `tests/test_gateway_async.py` — allow / deny / review,
     resource-arg binding, kwarg stripping, `apg_resource` override,

     web→email exfiltration scenario denied at send.
   - `tests/test_mcp_adapter_async.py` — discovery, prefixing, allow +
     deny, taint propagation, resource binding, audit integration,
     misshapen-session errors.
5. Verify `pytest -q` and `ruff check .` are green; update CHANGELOG and
   ROADMAP; commit; push.
