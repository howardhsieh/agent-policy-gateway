# Work log — 2026-06-03

## Roadmap item

**R24. `apg policy diff <old> <new>` CLI subcommand.** Compare two policy files by
*decisions*, not text: evaluate a matrix of synthetic tool calls against both
policies and report scenarios whose first-match outcome changed.

## Plan

- Add a `diff` subcommand to the existing `apg policy` argparse tree in `cli.py`.
- Synthesize the scenario matrix from both policies' selectors: each rule's
  selector is concretized (fnmatch globs -> sample values, taint clauses -> a
  satisfying source set), plus one neutral baseline scenario; deduplicated.
- A user-supplied `--tool/--identity/--taint/--resource` (like `explain`)
  replaces the matrix with that single scenario.
- Decision = `(rule_id, action)` from `Policy.first_match`, or no-match; report
  scenarios where old and new decisions differ; print "no decision changes"
  when none do. Exit codes: 0 ok (changes or not), 2 missing file, 1 malformed.
- Tests in `tests/test_cli.py`: effect flip, added rule, removed rule,
  no-change, missing/malformed files, single-scenario mode, glob concretizer.
