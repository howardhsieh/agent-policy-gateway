## Design choice note
The roadmap item names "LangChain / LlamaIndex"; I implemented a single
duck-typed adapter whose name extraction and invocation cover both frameworks'
tool-object shapes, so the framework is never a runtime dependency. Named for
LangChain (the most widely used) but LlamaIndex `FunctionTool` shapes are
covered by the same duck-typing.
