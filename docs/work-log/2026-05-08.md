# Work log — 2026-05-08

## Roadmap item
**R12. Benchmark harness: measure overhead per call, throughput.**

## Plan

- Add a small standard-library-only timing harness as a public module
  `agent_policy_gateway.bench`, exposing a frozen `BenchResult` dataclass
  (name, iterations, warmup, mean/min/max/p50/p95/p99 in nanoseconds,
  ops/sec) and a `benchmark(name, fn, *, iterations, warmup)` runner
  that uses `time.perf_counter_ns()` per call so percentiles reflect
  real per-call latency rather than wall-clock-divided averages.
- Provide four canonical scenarios that span the gateway's hot paths so
  overhead and throughput can be compared apples-to-apples:
  raw-call (no gateway), gateway-allow (`Gateway.execute` + `wrap_tool`
  with a permissive policy), gateway-deny (selector matches → DENY
  raises `PolicyDenied`, exception fast-path), and gateway-allow+audit
  (in-process no-op `AuditWriter`, isolating the audit call cost from
  the JSON+fsync cost of the JSONL writer).
- Expose `run_default_suite(iterations, warmup)` that returns the four
  results in a stable order, plus `format_results_table(results)` for a
  fixed-width text rendering and `results_to_json(results)` for machine
  consumption.
- Wire a `apg-bench` console script via `[project.scripts]` ->
  `agent_policy_gateway.bench:bench_main`. Flags: `--iterations N`,
  `--warmup N`, `--json` (stdout), `--scenario name` (repeatable).
- Tests in `tests/test_bench.py`:
  smoke runs with small N complete quickly; `BenchResult` percentile
  ordering and types invariants; each scenario actually exercises its
  contract (deny scenario raises through the timed callable, audited
  scenario records once per iteration); `format_results_table` and
  `results_to_json` shape checks; CLI entry point handles `--json`,
  `--scenario`, and `--scenario unknown` (exit 2).
- Files added: `src/agent_policy_gateway/bench.py`,
  `tests/test_bench.py`, plus `docs/benchmarks.md` describing the
  harness and how to run it. `pyproject.toml`, `__init__.py`,
  `README.md`, `mkdocs.yml`, `docs/design.md`, `CHANGELOG.md`,
  `ROADMAP.md` updated.
