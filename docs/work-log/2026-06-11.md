  file). Malformed lines still exit 3; exit 2 is impossible for stdin.

- `tests/test_audit_stats.py`: add a `TestAuditStatsStdin` class that pipes a log
  via `monkeypatch`/`io.StringIO`, asserting (1) `-` produces the same summary as
  the file path, (2) `<stdin>` header, (3) malformed piped line exits 3, (4) JSON
  mode over stdin.
